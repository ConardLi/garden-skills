# 教学原子映射表

> 本文档是 YJango 学习理论的教学执行映射。所有学习内容都由三个原子操作组合而成。

---

## 三个教学原子

### 🧩 判别原子

**定义**：建构判别模型——输入(现象)→输出(类别标签)
**YJango 原始概念**：见 raw_15「判别与联结」

| 维度 | 内容 |
|------|------|
| 核心问题 | "这是什么？" "属于A还是不属于A？" |
| 建构方法 | **自下归纳**（给 ≥5 正例找共性）← 缺例子时用<br>**旧异重组**（拆定义成积木块，逐块对比）← 有好的定义时用<br>**词络约束**（多语境暗示，体会共性）← 语言学含义时用<br>**渐构示范**（演示如何做判别，让用户模仿）← 教策略/元认知时用<br>**单义判别**（同一个词的不同语境，体会含义边界）← 语言学单义时用 |
| 方向 | 自下/自上可选，取决于材料是否有定义 |
| 材料类型 | 正例 + 反例 + 边界例（或：多段语境语料） |
| 检验方法 | 苏格拉底判别（给未见实例，判断属于/不属于 + 为什么）<br>——语言学中：单义判别验证（在未例句中判断是否是某含义） |
| 完成标志 | 能正确判别未见实例 |
| 精化方向 | 相近概念辨析（区分A与相似的B/C） |

### 🔗 联结原子

**定义**：建构联结模型——输入(概念)→输出(另一概念)
**YJango 原始概念**：见 raw_15「处理模型/联结模型」

| 维度 | 内容 |
|------|------|
| 核心问题 | "它们有什么关系？" "如果变X，Y会怎样？" |
| 建构方法 | **判联分析**（给关系图 + ≥3 组对比案例）← 通用<br>**自上应用**（给规则→执行，蕴含联结关系）← 技能中的"什么条件对应什么操作"<br>**示例：** 先展示完整关系图，再用不同案例比较关系是否成立 |
| 方向 | 自上为主，从关系描述推具体案例 |
| 材料类型 | 关系图 + 对比案例（或：规则说明 + 执行示例） |
| 检验方法 | 新情境推理（给未见场景，推理结果）<br>——技能中：外显分析（边做边说出"为什么这一步这么做"） |
| 完成标志 | 能推理新情境下的关系 |
| 精化方向 | 变条件推理（改变输入变量，推输出变化） |

### ⚡ 内隐原子

**定义**：将外显模型自动化——从有意识执行到无意识反应
**YJango 原始概念**：见 raw_20「内隐学习」

| 维度 | 内容 |
|------|------|
| 核心问题 | "做得快不快？" "需不需要想？" |
| 建构方法 | **外显执行**（按规则做，边做边说过程）← 通用起点<br>**重复练习**（同一动作反复执行，至流畅）← 固化阶段<br>**示例：** 先拆解步骤→用户按步骤做→逐渐减少提示→不提示→干扰中执行 |
| 方向 | 自上→内隐（从外显规则到自动执行） |
| 材料类型 | 练习场景题（正常·边界·干扰各型） |
| 检验方法 | 限时执行 / 干扰环境执行 |
| 完成标志 | 无需意识参与，自动完成 |
| 精化方向 | 复合应用（和另一原子组合使用，如多任务执行） |

## 学习场景 → 原子组合

用户面对的是学习场景（"我在学XX"），引擎内部映射为原子组合：

| 用户场景 | 原子组合 | 说明 |
|---------|---------|------|
| 学一个概念（"什么是编码"） | 判别 | 只需建一个判别模型 |
| 学一个命题（"反射定律"） | 判别(变量) → 联结(关系) | 先建变量的判别模型，再建关系的联结模型 |
| 学一个技能（"焊接"） | 判别(步骤+工具) → 联结(顺序) → 内隐(重复) | 三步完整走 |
| 学一个语言（"英语单词"） | 判别(含义) + 判别(符号) + 联结(指代) + 可选内隐(发音) | 多个判别模型堆叠 |
| 学一种策略（"调试方法"） | 判别(场景类型) → 联结(场景→策略) → 可选内隐 | 元层面的判别+联结 |

**教学执行规则**：
- 有 → 箭头的：按顺序执行，前一个未达标不进后一个
- 用 + 连接的：可以并行执行，不分先后

## 诊断表（用于决策树第二步的病因细化）

此表不单独使用。在统一回退决策树（execution-standards.md#4）执行到"首次没过→诊断病因"步骤时，用此表细化诊断：

| 检验表现 | 判联分析诊断 | 回退到 |
|---------|------------|--------|
| 判断不准 | 判别原子没建好 | Step ⑤ 换材料 / Step ⑥ 换方式 |
| 能判断但推不出新情境 | 有判无联（判别已有，缺联结） | 补联结原子建构 |
| 能说清但做不出 | 有联无判（联结已有，缺内隐） | 补内隐原子练习 |
| 只会用术语解释 | 言存义空（只有语言标签，无判别模型） | Step ⑤ 加正反例重建判别 |
