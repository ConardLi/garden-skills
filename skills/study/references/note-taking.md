# "整理"指令 — 学习笔记生成指南

## 触发

用户说："整理"、"保存进度"、"记笔记" 时执行。

## 自动收集以下信息

1. **主题与领域**（当前在学的内容）
2. **原子构成**（涉及哪些原子：判别/联结/内隐）
3. **当前 Step 位置**（渐构八步中的哪一步）
4. **已建立的模型**（判别模型/联结模型，列举具体内容，标注由哪个原子完成）
5. **已通过的检验**（用户答对的问题记录，标注对应原子）
6. **待完成**（当前原子的工作流阶段及未完成部分）
7. **下一步计划**（计划进入的下一个原子或阶段）
8. **备注**（用户常见错误、难点、偏离方向等值得记录的信息）

## 笔记文件格式

```markdown
---
topic: 电机学习 - 伺服控制
atoms: 判别 + 联结
step: 7
saved_at: 2026-05-10
next: 联结原子—电子齿轮比的关系推理
---

## 背景基线
- 用户基础：有电路基础，知道直流电机正反转
- 目标：选型、接线、驱动控制
- 拆组方式：自上而下

## 已建立的模型

### 电机三类型的判别模型（🧩 判别原子 ✅）
- [x] 直流：通电就转，控制转速，无反馈
- [x] 步进：脉冲控制角度，开环，轻负载稳定下精度高
- [x] 伺服：脉冲控制角度，闭环（编码器），负载突变可补偿
- 核心判别：开环 vs 闭环

### 伺服系统的联结模型（🔗 联结原子 ✅）
- [x] 三大件关系：PLC/控制器 → 驱动器 → 电机
- [x] 接线对应关系：编码器线(大黑头)、动力线(多P)、电源线(多P)、控制信号线(小插头)

### 伺服三信号（🧩 判别原子 — 待检验）
- [ ] SON（使能）：拉高后锁定轴，准备响应脉冲
- [ ] PLS（脉冲）：每个上升沿走一步
- [ ] DIR（方向）：高/低电平控制正反转

## 已通过的检验

- [x] 持续运行变速场景选直流电机（判别原子）
- [x] 步进vs伺服本质区别：开环 vs 闭环（判别原子，修正过）
- [x] 伺服没接控制信号线不会转（联结原子）

## 待完成

### 🧩 判别原子—电子齿轮比
- 当前阶段：判别建构（已完成）
- 待完成：判别检验（五维饱和）
```

## 保存位置

- 保存到 Obsidian vault 根目录（即 `.claude` 所在目录）下的 `学习记录/` 文件夹
- 文件命名：`{主题简短拼音或英文}-{日期}.md`
- 示例：`电机学习-2026-05-10.md`
- 如果 `学习记录/` 不存在，自动创建

## 恢复教学

当用户说"继续学习XX"或提供文件名时：
1. 在 `学习记录/` 下找到对应笔记
2. 读取 `atoms`、`next`、`已建立的模型` 和 `待完成`
3. **跨日先复检**（execution-standards.md#11.5）
4. 输出一句："已恢复 [topic] 教学，当前涉及原子 [atoms]，上次学到 [next]，继续这部分。"
