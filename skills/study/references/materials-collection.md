# 学习材料搜集策略

在 Step ⑤ 执行"分析材料"前，先询问用户是否需要搜索权威学习材料。

---

## 询问模板

```
要开始学 [主题] 了。需要我先帮你搜集一些学习材料吗？

📚 书籍     — 适合系统理解概念和体系
💻 GitHub   — 适合看实际项目架构和代码
📄 论文     — 适合精确定义和前沿知识
🎬 高播放视频 — 适合直观演示和操作

你可以选一个或多个，也可以说"不需要，直接开始"。
```

---

## 搜索策略（按来源区分）

搜同一主题时，按来源拆成独立搜索，每个搜索有明确的**用途目标**。

| 来源 | 搜索词构造 | 质量筛选词 | 用途目标 | 示例 |
|------|-----------|-----------|---------|------|
| 📚 **书籍** | `主题 + 书籍/书单 + 推荐/经典/入门` | 豆瓣高分、经典、系统性 | 提取概念体系和关系框架 | `上位机 WPF 书籍 推荐 经典` |
| 💻 **GitHub** | `主题 + 开源 + 完整项目` | site:github.com, Gitee，限定 Star 排序 | 找可参考的完整项目架构和源码 | `WPF 上位机 开源 完整 MVVM Modbus site:github.com` |
| 📄 **论文** | `主题 + 论文 + 综述/定义/协议` | 高引用、顶会/顶刊、作者权威 | 寻找精确定义和前沿技术方案 | `modbus TCP 实时性 论文` |
| 🎬 **视频** | `主题 + 实战/从零/完整教程 + B站` | 高播放量、最新年份、系统性 | 找可以跟着做的项目教程 | `WPF 上位机 从零开发 实战 B站` |

**关键原则**：同主题按资源类型拆成 3-4 个独立搜索，每个搜索明确用途。不混搜。

---

## 双向映射：来源 ↔ 学习类型

### 来源 → 学习类型

| 来源 | 最适配的学习类型 | 推荐时机 |
|------|----------------|---------|
| 📚 **书籍** | 命题学习、策略学习 | 用户想"系统学"、"从零学"时 |
| 💻 **GitHub** | 技能学习、编程域 | 涉及编程/框架/工具时自动推荐 |
| 📄 **论文** | 概念学习、研究型学习 | 用户想"深入理解"、学术概念时 |
| 🎬 **高播放视频** | 技能学习、语言学习 | 涉及操作演示、语言学习时 |

### 学习类型 → 搜索策略

| 学习类型 | 优先搜索来源 | 搜索要点 |
|---------|------------|---------|
| 🧩 判别（学概念） | 论文 > 书籍 > 视频 > GitHub | 搜精确定义、对比材料、同义词辨析 |
| 🔗 联结（学关系） | 书籍 > 论文 > 视频 > GitHub | 搜关系图、流程、案例对比、规则说明 |
| ⚡ 内隐（学技能） | 视频 > GitHub > 书籍 > 论文 | 搜操作演示、步骤拆解、练习素材 |

---

## 搜索执行流程

1. **用户选择来源**（可多选）
2. **按来源策略分别搜索**（WebSearch），每来源搜索 1-2 次
3. **整理结果**：标注来源、推荐理由、与用户目标的匹配点
4. **呈现给用户**：按来源分组展示，附链接
5. **将材料纳入教学**：提取的内容作为 Step ⑤ 分析对象和 Step ⑦ 执行素材

## 市场验证（求职场景补充）

当学习目标是"找工作"时，**在执行材料搜集前，先搜索该岗位的市场技能要求**（BOSS直聘/智联/前程无忧），用市场要求指导搜索重点和原子优先级排序。

操作步骤：
1. 搜 `主题 + 招聘/面试 + 技能要求 + 202X`
2. 提取高频出现的 P0/P1 技能列表
3. 用市场权重调整原子拆组的优先级
4. 再按优先级搜对应的学习材料

## 来源特长说明

### 📚 书籍
- 擅长：概念体系、完整方法论、学科入门
- 关注：经典教材、豆瓣高分、领域共识
- 使用：提取核心框架作为 Step ⑥ 分析依据，关键章节作为建构材料

### 💻 GitHub
- 擅长：编程技能、框架学习、最佳实践
- 关注：Star 数高、活跃维护、文档完善、有完整架构
- 使用：README 用于概念理解，示例代码用于技能练习，项目架构用于联结分析

### 📄 论文
- 擅长：概念边界精确定义、前沿趋势
- 关注：高引用、顶会/顶刊、作者权威
- 使用：定义部分用于判别原子建构，实验结论用于验证

### 🎬 高播放视频
- 擅长：操作演示、项目跟做、直观理解
- 关注：高播放量、从零到完成、结构清晰、带源码
- 使用：从头跟做完整项目，视频中的操作步骤作为内隐原子的外显示范
