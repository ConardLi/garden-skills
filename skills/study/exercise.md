# 练习引擎（Exercise Engine）

被 **Step ⑦（执行）** 和 **Step ⑧（验证）** 调用。不单独使用。

---

## 原子工作流

每个教学原子有自己的工作流，阶段数不统一。

### 🧩 判别原子工作流

```
┌─ 判别建构 ──→ 判别检验 ──→ 判别精化 ──→ ✅ 完成
   ↑ 换材料       ↑ 不饱和       ↑ 辨析不过
   └─────── 回退 ─┴──────┴───────┘
```

**判别建构**：给 ≥5 正例 + ≥3 反例 + 边界例，让用户体会共性差异
**判别检验**：苏格拉底判别——给未见实例，判断属于/不属于 + 为什么
  - 过关标准：**五维饱和**（见 execution-standards.md#11）
  - 否则同建构换材料重练
**判别精化**：相近概念辨析——给相似概念A/B/C，让用户区分边界
  - 过关标准：能准确区分三个以上相近概念，解释边界依据

### 🔗 联结原子工作流

```
┌─ 联结建构 ──→ 联结检验 ──→ 联结精化 ──→ ✅ 完成
   ↑ 换材料       ↑ 推不出       ↑ 变条件错
   └─────── 回退 ─┴──────┴───────┘
```

**联结建构**：给关系图 + ≥2 组对比案例，让用户理解变量间的映射
**联结检验**：新情境推理——给未见场景，推理输入改变后输出的变化
  - 过关标准：连续 3 题不同场景全部推理正确
  - 否则同建构换案例重练
**联结精化**：变条件推理——改变输入变量，追问输出如何变化
  - 过关标准：能准确描述条件和例外情况

### ⚡ 内隐原子工作流

```
┌─ 外显执行 ──→ 重复固化 ──→ 干扰验证 ──→ ✅ 完成
   ↑ 重新讲解     ↑ 未自动化     ↑ 干扰中断
   └─────── 回退 ─┴──────┴───────┘
```

**外显执行**：用户按规则执行操作，边做边说出过程（外显分析）
  - 过关标准：能在无提示下完整执行一次
**重复固化**：多次重复练习，直到速度和流畅度明显提升
  - 过关标准：不再需要意识思考步骤
**干扰验证**：在干扰/限时/多任务环境下执行，测试自动化程度
  - 过关标准：干扰下仍然能正确执行

## 原子组合教学

当一个学习场景涉及多个原子时，按**依赖顺序**执行：

```
判别(地基) ──→ 联结(上层) ──→ 内隐(固化)
```

- 判别未过 → 不能进入联结
- 联结未过 → 不能进入内隐
- 并行原子（用 + 连接的）可以穿插执行

**示例——学"电子齿轮比"**：
```
1. 判别原子：电子齿轮比是什么？正反例对比 → 五维检验 → ✅
2. 联结原子：分子/分母的关系 → 新情境推理 → 变条件 → ✅
3. 内隐原子：（如果需要动手配置）外显操作 → 重复 → 干扰验证 → ✅
```

## 流转规则

- 当前阶段未过 → 同阶段换工具/材料重练
- 持续不过 → 退回同原子的前一阶段
- 判别不通过 → **禁止**进入联结原子（依赖关系）
- 原子所有阶段通过 → ✅ 该原子完成
- 场景涉及的所有原子完成 → ✅ 当前原子完成，可进入下一原子
