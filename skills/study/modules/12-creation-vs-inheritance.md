# 十二、创验学习 vs 继承学习

> **核心区分**：模型是自己创造的，还是从他人处继承的。

## 学习分类的五个视角

为全面分析学习，课程引入**五个视角**组合运用：

| 视角 | 核心问题 |
|------|---------|
| **继承视角** | 模型从哪来？（自己创造 vs 从他人继承） |
| **方向视角** | 建构方向如何？（从具体到抽象 vs 从抽象到具体） |
| **语言视角** | 语言参与程度？ |
| **复用视角** | 知识应用范围？ |
| **系统视角** | 知识组织方式？ |

本模块聚焦**继承视角**。

## 完整对比

| 维度 | 创验学习 | 继承学习 |
|------|---------|---------|
| **流程** | 搜集数据→抽象概念→建构模型→验证模型 | 获取材料→继承概念→继承模型→比对模型 |
| **数据来源** | 与环境交互 | 他人学习材料 |
| **失败后** | 返回数据/概念/猜想迭代 | 调整理解或获取更多材料 |
| **优点** | 理解深刻，知识灵活 | 效率高，方向明确 |
| **缺点** | 效率低，易以偏概全 | 易失去过程理解 |

## 关键辨析

| 易混淆 | 纠正 |
|--------|------|
| 创验学习 = 科研 | 科研是创验学习的子类。创验不要求创造"从未有过的模型"，五岁小孩学会"一闹父母就妥协"也是创验学习 |
| 继承学习 = 被动学习 | 继承需要主动加工材料，是被动复述无法完成的建构过程 |
| 继承学习 = 接受学习 | 接受学习需老师参与；继承可自行主导（如偷师） |

## 社会视角 vs 认知视角

**社会视角**（大众常用）：学习 = 向他人获取知识或经验。
**认知视角**（课程采用）：学习 = 建构可泛化模型的行为。

社会视角会产生**概念遮蔽**：将学习方法窄化为"如何记住他人的分享"，把创验学习排除在外。
认知视角关注人脑先天机制，创验和继承都是"建构可泛化模型"，所以都属于学习。
