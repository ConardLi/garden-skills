# 十、内隐 vs 外显 / 意识 vs 自主

## 核心区分

| | 意识处理 | 自主处理 |
|---|---|---|
| **注意力** | 需要 | 无需 |
| **控制** | 可意识控制 | 自动运行 |
| **速度** | 慢 | 快 |
| **容量** | 有限 | 无限 |

## 内隐记忆 vs 外显记忆
- **内隐记忆**：无法用语言表达的、无意识的记忆
- **外显记忆**：可被意识表达的、有意识的记忆

## 内隐模型 vs 外显模型
- **内隐模型**：自主建构、无法意识详细运作过程的模型
- **外显模型**：可描述、可分析的模型
- 依赖中文的外显模型 → 导致中式英语和执行缓慢

## 自主化过程
有意识练习 → 重复 → 自动化 → 无需意识

## 应对模式
大脑自主系统先归类，再用该类的应对方式处理。
- **应对模式误判**：大脑自主系统对情况错误归类
- **自主编码遮蔽**：自主分类导致看不到其他可能性
- **知识的诅咒**：知道后难以想象不知道的状态

**关联方法：** 刻意练习 (Ericsson) — 有明确目标 → 专注执行 → 即时反馈 → 不断突破舒适区。这是"从外显到内隐"的实现路径：先将外显知识分解为可练习的子技能，在反馈中反复修正，直到自动化。关键在于"即时反馈"——没有反馈的重复只会固化错误，不会自动化正确。
