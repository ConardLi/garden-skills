# 十八、学习框架总览（9个Level）

```
Level 0: 元认知 → 规范性学习 vs 凭感觉学习
Level 1: 生存依赖 → 经验预测 vs 模型预测
Level 2: 概念世界 → 概念 = 判别模型，五组概念
Level 3: 分类 → 类别固有观，划分的支撑
Level 4: 信息推测 → 知识的数学描述
Level 5: 判联分析 → 判别 + 联结
Level 6: 语言世界 → 语言 = 概念代理
Level 7: 学习 vs 记忆 → 严格区分
Level 8: 综合应用 → 分析任何学习问题
```
