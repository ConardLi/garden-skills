# 二十、方向视角：自下学习 vs 自上学习

> **核心区分**：知识建构的方向——从具体现象到抽象规律（自下），还是从抽象规律到具体现象（自上）。

## 知识的下上结构回顾

```
上层（共象层）= 抽象规律，预测未见情况的工具
下层（对象层）= 具体对象/事件，要预测的问题
```

## 自下学习（Bottom-up）

**定义**：从下层具体现象出发，向上建构（连接到）上层抽象规律。

**示例**：拽枯草——经历三根/五根/七根的具体事件 → 抽象出"根数""费力程度"概念 → 归纳出"根数越多越费力"。

**特点**：易以偏概全，需反复试错验证；天然容易理解（演化本能）。

## 自上学习（Top-down）

**定义**：从上层抽象规律出发，向下建构（连接到）下层具体现象。

**示例**：小伙伴从你告诉他的"根数越多越费力"出发 → 连接到自己生活中的现象 → 做出草绳秋千。

**特点**：继承他人验证成果，方向明确；但抽象表述有理解门槛。

## 纠正常见误解

| 误解 | 纠正 |
|------|------|
| 自下 = 直接经验 | 也可以基于他人告知的间接经验（告知具体事件，自己归纳） |
| 自下 = 归纳学习 | "自下"指方向非思维方式，也可运用演绎和类推 |
| 自上 = 演绎学习 | "自上"指方向非思维方式 |
| 自上无法创造新知识 | 演绎可将既有知识中隐含结论显性化，对有限的人脑而言就是新知 |

## 三方面优劣

| 维度 | 自下学习 | 自上学习 |
|------|---------|---------|
| **普遍性** | 易以偏概全，需反复试错 | 继承验证成果，仅当准确理解时才发挥优势 |
| **理解性** | 天然容易（基于具象经验） | 抽象表述四大障碍：缺对象、需训练、需领域概念、鸡同鸭讲 |
| **脱节性** | 易丢失上层（经验坍缩） | 易丢失下层（悬空知识/抽象空转） |

## 双重结构坍缩

两种学习方式都容易导致**结构坍缩成单层**：

- **自下丢失上层**：把具体例子原封不动记忆 → 只能解决和例子一模一样的问题
- **自上丢失下层**：抽象概念成为空中楼阁 → 能背术语但举不出例子
- **判断是否真学会的最直接方式：跟对方要例子。**

## 与创验/继承视角的关系

**自下/自上** 和 **创验/继承** 是两个独立的分类维度，不可混为一谈：

| | 创验学习 | 继承学习 |
|---|---------|---------|
| **自下学习** | 从交互中自己归纳出规律（如拽枯草故事中的"你"） | 听别人讲具体事例，自己归纳出规律 |
| **自上学习** | 用既有知识演绎推导出新知识（如"偶数+奇数=奇数"的推理） | 听别人讲抽象规律后应用到具体（如小伙伴做秋千） |

**关键结论**：
- 自下 ≠ 创验。继承学习也可以自下（他人告知具体事例，自己归纳）
- 自上 ≠ 继承。创验学习也可以自上（用已有知识推出新结论）
