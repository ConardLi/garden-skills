# 九、学习 vs 记忆（严格区分）

## 对比

| | 学习 | 记忆 |
|---|---|---|
| **课程定义** | 获取预测未见情况能力的行为 | 获取重现已见情况能力的行为 |
| **标准定义** | 建构可超越已有经验普遍性（可泛化）模型的行为 | — |
| **对象** | 模型（通用判别规则） | 经验（具体已见） |
| **能力** | 处理未见情况 | 再现已见情况 |
| **结构** | 跨两层（下上结构） | 同一层 |
| **验证** | 能否判别未见实例 | 能否回忆起已见 |

## 学习定义的进化（V1~V4）
- V1：直接经验的实践交互
- V2：基于语言引导的实践
- V3：语录记忆+实践
- V4：课堂听讲+记忆（校园片面理解）

## 常见误区
- **行为目标混淆**：把记忆目标当作学习目标
- **验证手段混淆**：用记忆的方式来验证学习成果
- **"背板" vs "建模"**：记忆固定套路 vs 建构可泛化模型
- **E类 vs T类学生**：勤奋但方法错（记忆代替学习）vs 无意中掌握了模型建构

## 神经科学证据
- **H.M.病人**：能学新技能但无法形成新记忆——学习与记忆分离
- **吉姆·皮克**：照相式记忆但学习能力低——反向证明

## 适用场景
- 某些情况下记忆更有效率（效率成本差异）
- 根据具体情况选择记忆或学习
- 建构模型只是实现学习的手段，不是目的本身

**关联方法：**
- 检索练习 (Roediger & Karpicke) — 主动从记忆中提取信息比重复阅读效果高2-3倍。适合巩固已建构的模型，同时也是检验学习成果的可靠手段（能否准确提取而不是能否认出）。
- 间隔重复 (Ebbinghaus) — 在遗忘临界点复习，最大化记忆效率。YJango做了"学习vs记忆"的区分，这两个方法分别优化"学习的泛化检验"和"记忆的效率"。
