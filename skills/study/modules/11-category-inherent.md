# 十一、类别固有观与划分理论

## 类别固有观
认为类别天然存在于自然界中的错误观念。

**派生信念链条**：
类别是客观存在的 → 概念不会变 → 不存在更好的分类 → 限制创新

**持续存在的原因**：教育环境、缺乏先验知识

**负面影响**：限制创造力、兼容力、理解力

## 划分的支撑
- **相互依存**：概念相互依赖，互为定义
- **概念组模型**：如"发信者/收信者/消息"——三个概念共同创生、共同消解
- **先天概念**：饿/非饿、痛/非痛、我/非我、利/害
- **划分的支撑三角**：现象群 ← 判别模型 → 概念 ← 现象群

## 反映的客观
- 概念反映的是**对象间的关系**，不是对象本身
- 客观性在于关系的一致反映，不在于概念本身是否"真实"

## 理论不唯一
- 同一现象可以用不同概念体系描述
- **中介概念**：没有直接物理对应的概念（如"力""时间""能量"）
- **概念收敛**：多种约束导致不同文化对相似概念的划分趋同

**关联方法：** 概念转变理论 (Posner) — 改变已有错误概念需要四个条件：(1)对现有概念不满 → (2)新概念可理解 → (3)新概念看似合理 → (4)新概念能解决更多问题。当学习者被"类别固有观"束缚时，用这四个条件诊断卡在哪一步——多数人卡在(1)，他们尚未意识到现有分类方式有问题。
