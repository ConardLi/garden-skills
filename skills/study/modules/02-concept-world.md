# 二、概念世界

## 概念 = 判别模型

**概念** = 一个**判别模型**，它将所有可能的现象划分为两个类别：属于该概念（正概念）和不属于该概念（负概念）。

**核心认识**：
- 学习概念 ≠ 记住概念的定义
- 学习概念 = 建构出能判断"属于/不属于"的判别能力
- 验证标准：能否正确判别**从未见过**的实例
- 学习是渐构过程：判别规则不够通用 → 搜集新材料 → 重建规则

## 概念建构流程（规范化）
1. **判断学习类型** → 是否为概念学习
2. **确立学习目标** → 建构"通用判别规则"
3. **搜集学习材料** → 具体样例 + 他人描述
4. **正确运用材料** → 比较样例找共有属性
5. **验证学习成效** → 用未见情况测试，迭代规则

## 五组概念（核心概念系统）

**第一组 — 两个层面**：
- **具象层**：抽象前要处理的对象所在的层面
- **抽象层**：抽象后得到的类别所在的层面

**第二组 — 五个元素**：
- **对象**：实际要预测和处理的事物
- **内涵**：从已见对象中提取的共有属性（判别条件）
- **概念**：抽象层面的类别（可代表一类事物）
- **判别模型**：通用判别规则，根据内涵判别归属
- **外延**：某概念在具象层对应的所有对象

**第三组 — 三个行为**：
- **抽象（动词）**：从已见中提取共有属性的行为
- **判别（动词）**：应用判别模型判断归属的行为
- **具象（动词）**：从概念联想具体对象的行为（抽象的反过程）

**第四组 — 两种对象**：
- **已见**：已见过的对象，用于抽象
- **未见**：未见过的对象

**第五组 — 两个量词**：
- **一个**：对象和概念的量词
- **一类**：层和外延的量词

**重要辨析**：
- 概念 ≠ 外延（概念在抽象层，外延在具象层）
- 概念 ≠ 内涵（概念是标签，内涵是属性）
- 概念 ≠ 判别模型（概念是结果，判别模型是规则）
- 判别 ≠ 具象（判别是归类行为，具象是联想行为）
- 一分必出二：创造概念必然同时产生正概念和负概念

## 概念学习 = 学习判别方式
- 给你任意现象，你都能判断它是这个概念还是不是
- **绝对视角的非概念**：未被归类之前的原始物理信号
- **相对视角的非概念**：已被归为概念的事物，在再次被归类时也是"现象"
- **现象-判别模型-概念的三元组**：三者构成关系结构，必须一同理解

**关联方法：** 概念获得模型 (Bruner) — 教师只给正反例，学习者自己归纳概念。操作流程：(1)呈现正例和反例 → (2)学习者提出假设 → (3)更多例子验证/修正假设 → (4)命名概念。与YJango的"概念=判别模型"完全同构，但提供了更具体的操作步骤。
