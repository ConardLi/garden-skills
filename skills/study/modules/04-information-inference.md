# 四、信息推测框架

## 第一性原则
**学习的第一原则：明确输入输出。** 知识本质是"从输入到输出的映射关系"。

## 信息推测框架
```
输入空间 → 映射关系（知识） → 输出空间
```

## 关键概念
- **输入空间**：所有可能输入的集合
- **输出空间**：所有可能输出的集合
- **变量**：输入变量和输出变量
- **映射**：输入到输出的对应关系
- **经验推测**：从输入到输出的直接记忆和重现
- **模型推测**：从输入经过模型泛化到输出的过程
- **知识 = 映射关系**：知识被定义为输入空间到输出空间的映射关系

## 判别模型结构
```
输入(任意现象) → 判别模型 → 输出(此概念/非此概念)
```
- **一般输入空间**：实际的输入范围
- **标准输入空间**：理论上的完整输入范围
- **判别能力**：能否正确判断类别
- **具象能力**：能否列举属于该类别的具体现象

## 联结模型结构
```
概念A的外延 → 联结模型 → 概念B的外延
```
- **联结模型** = 概念之间映射关系的模型
- **三段论关系**：大前提 + 小前提 → 结论
- **判别+联结组合**：判别模型识别概念，联结模型推断关系

**关联方法：** PQ4R阅读法 — Preview(预览) → Question(提问) → Read(阅读) → Reflect(反思) → Recite(复述) → Review(复习)。其中PQ两步强制你在阅读前就明确输入(主题)和输出(要回答的问题)，符合"学习的第一原则：明确输入输出"。
