# 八、五大误区（常见学习陷阱）

## 1. 记言代学
把记忆语言当作学习——这是最常见的误区。

## 2. 言存义空
语言存在但意义空泛（能说出术语但不懂含义）。

## 3. 义存言空
意义存在但无法用语言表达（这是正常的，内隐知识的特征）。

## 4. 知识坍缩（三种形式）
- **下层丢失**：失去了与具体现象的联系
- **上层丢失**：失去了抽象概括
- **下上层错配**：抽象层和具象层不匹配

## 5. 现实脱离
概念与现实脱节。

**正确学习顺序**：理解语义 → 组织材料 → 建构模型 → 建立指代

**关联方法：** 费曼技巧 — 选概念 → 教给零基础的人 → 发现卡点 → 回头重学 → 简化类比。专门用来检测"言存义空"和"记言代学"。如果你讲的时候需要背术语而非自然解释，说明还没建构出判别模型。卡住的地方就是你的"知识坍缩"缺口。
