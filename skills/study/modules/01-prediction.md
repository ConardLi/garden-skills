# 一、预测：学习的核心能力

## 经验预测 vs 模型预测

| | 经验预测 | 模型预测 |
|---|---|---|
| **层级** | 同一个层级 | 跨两个层级 |
| **对象** | 已见情况 | 未见情况 |
| **方式** | 记忆和回忆 | 用抽象规律泛化 |
| **结构** | 只有对象层 | 形成下上结构 |
| **能力** | 认出见过的东西 | 处理没见过的东西 |
| **局限** | 无法应对未见情况 | 依赖假设成立 |

## 经验预测的机制
- **流程**：经历经验 → 保存经验 → 假设"过去=未来" → 重现预测
- **最大局限**：只能预测已知，不能预测未知
- **获取代价**：每次尝试都可能付出生命代价
- **已见牢笼**：如果只有经验预测，每个新情况都是未知危险

## 模型预测的机制
- **流程**：积累经验 → 抽象概念 → 归纳模式 → 建构模型 → 验证假设 → 更新模型 → 用模型预测
- **核心能力**：压缩能力（将多经验压缩为一个规律）+ 泛化能力（预测未见情况）
- **关键概念**：
  - **抽象**：从众多事物中抽取共有特征、剔除非共有特征后创造概念
  - **表征**：用抽象属性替代具体事物（"替代品"）
  - **归纳**：找出两类信息之间在多个情况中共有关系的行为
  - **模型**：对现实简化后的关系表述
- **局限**：无法保证模型永真，只能保留尚未泛化失败的模型

## 微观处处未见（认识论基础）
- 纯自然状态下遇到完全相同的情况概率为零
- 大脑处理流程：物理信号 → **感觉**(神经电信号) → **感知**(概念/宏观现象)
- **感觉 vs 感知**：感觉将物理信号转为神经信号，感知将神经信号判别为概念
- 我们感知到的"宏观现象"是大脑对微观信号的**预测结果**

**关联方法：** 第一性原理思考 — 不依赖类比或现有答案，而是归因到最基础的不可再分的原理，再重新推导。这与模型预测的"从经验中抽象出规律"同构，区别在于第一性原理对"底层假设不可再分"的要求更严格。
