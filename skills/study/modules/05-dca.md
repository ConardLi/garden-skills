# 五、判联分析（Discriminant-Connection Analysis）

## 判别模型 vs 联结模型
| | 判别模型 | 联结模型 |
|---|---|---|
| **功能** | 判断"是什么" | 推断"什么关系" |
| **输入** | 任意现象 | 概念的外延 |
| **输出** | 类别标签 | 另一概念的外延 |
| **类比** | 分类器 | 推理规则 |

## 三种失败模式
1. **有判无联**：有判别能力但缺乏联结关系（知道是什么但不知道关系）
2. **有联无判**：有联结关系但缺乏判别能力（知道关系但无法判断是什么）
3. **判联错配**：判别和联结不匹配（用错了地方的推理）

## 核心诊断工具
- 对象层丢失：失去了与具体现象的联系
- 定义 vs 判别：定义不等于判别能力
- 内隐判别模型 vs 外显判别模型

**关联方法：** 类比推理 (Gentner) — 结构映射理论：将源领域的关系结构映射到目标领域。本质上是"建立跨领域联结模型"的操作方法。步骤：(1)识别源领域的关系结构 → (2)找到目标领域的对应元素 → (3)逐层映射。用于训练"联"的能力。
