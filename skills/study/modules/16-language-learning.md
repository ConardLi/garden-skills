# 十六、语言学习专项模型

## 背单词拆解框架
```
学识义（建构含义的判别模型）
+ 学识字（建构文字符号的判别模型）
+ 学识音（建构语音符号的判别模型）
+ 练发音（建立含义→肌肉信号的对应）
+ 练写字/打字（建立含义→书写信号的对应）
+ 建指代（建立符号↔含义的指代关系）
```

## 听力误区框架
- 仅听一例（不充分）
- 期待速成（不现实）
- 听声本身（忽视意义对应）
- 意识操控（不应干预自主）

## 有效听力材料三条件
感官刺激 + 有效经验 + 新颖情况

## 单义判别训练
- **横拆 vs 竖拆**：单义判别（横向拆解）vs 背单词+语法（竖向拆解）
- **三步法**：选择含义 → 搜集例句 → 归纳比较

## 重要概念
- **可理解输入**：输入+对应输出
- **i+1原则**：略超当前水平
- **习得 vs 学得**：内隐建构 vs 外显规则
- **负迁移**：母语对二语学习的干扰
- **母语中心**：母语作为"操作系统"难以更换

**关联方法：** CLIL (Content and Language Integrated Learning) — 用目标语学学科内容，语言和意义同步获得，避免"为学语言而学语言"的脱离意义问题。沉浸式学习 (Krashen) — 大量可理解输入，自然习得而非刻意学得。
