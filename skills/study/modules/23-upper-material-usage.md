# 二十三、上料用法：如何用上层材料

> **核心**：语言让人能**超验思考**——把不同场景的概念汇聚重组，想象从未经历的事物。

## 方式一：旧异重组（显性用法）

**定义**：依照材料给出的组合方式，对已有概念进行重组建构出待学知识。

**形式**：新概念 = 差别特征 + 上层概念（属+种差）

**示例**："偶数是能被二整除的整数"——把"能被二整除"和"整数"这两个旧概念组合出"偶数"。

## 方式二：词络约束（隐性用法）

**定义**：借助代学词在语言中的隐性关系网络来建构概念。

**原理**：一个词存在于足够多样的语句中→受多重语法和常识约束→被赋予足够属性→大脑建构出判别模型。

**示例**：没查过字典也能在阅读中掌握词汇含义；读《论语》无"君子"定义却形成认识。

**与旧异重组的区别**：旧异重组有明确重组指示；词络约束依靠隐性关系网络。

**潜在风险**：语料源于说话者的认知结构→世界观在无形中传递。如"不好好学习长大只能送外卖"同时传递知识和价值观念。

## 自上学习的三个前提

| 前提 | 说明 |
|------|------|
| **掌握基础概念** | 解释链触底或循环时需自下学习兜底（如"左是右的反方向"需在现实中同时建构） |
| **掌握语法规则** | 能正确解析句子成分，必要时进行表达转换（如"头孢配酒有毒"需转换为"行为→后果"模型） |
| **明确行为目标** | 知道"学习=建构可泛化模型"，带着目标分析材料而非顺序阅读记忆 |

**无目标的典型后果**：读百科"熵"的介绍→记住了全文但感觉模糊→不知道自己没学会。
